# docs-web

## Getting Started

### 1.本地使用方法

#### 1.1 设置国内源地址
```
npm config set registry https://registry.npm.taobao.org
```

#### 1.2 安装依赖
```
npm install
```
#### 1.2 本地预览
```
npm start
```

#### 1.3 提交源码
```
git push
```

#### 1.4 构建打包
```
npm run build
```
